Export Data

Download your data and update on GitHub:

GitHub Update Instructions:

  1. Download the data.json file
  2. Go to your GitHub repository
  3. Open data.json file
  4. Click "Edit" button
  5. Replace all content with new JSON
  6. Click "Commit changes"
  7. Wait 1-2 minutes for updates to reflect
